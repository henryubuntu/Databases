Learn the videotutorial.

SQL (Structured Query Language) is a common tool for retrieving data from relational databases such
as SQL Server, MySQL, MariaDB, and PostgreSQL. This course provides an introduction to this core
programming language. Learn how to request data from a server, limit and sort the responses,
aggregate data from multiple tables with joins, and edit and delete data. Instructor Scott Simpson also
shows how to perform simple math operations and transform data into different formats.
